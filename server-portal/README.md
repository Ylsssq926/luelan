# Server Portal - 个人主站

展示AI工具链能力、产品经理技能和项目作品的个人主站。

## 技术栈

- React 18 + TypeScript
- Vite 5
- Tailwind CSS
- Lucide React Icons

## 开发

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build

# 预览生产构建
npm run preview
```

## 部署

构建后的文件在 `dist` 目录，可以部署到任何静态托管服务。

## 特性

- 响应式设计
- 现代化UI
- 项目展示
- AI认证展示
- 技术栈展示
- 个人介绍
